# RotoGolf
2D Golf game with gameplay around gravity

# Controls

Aim : Mouse cursor

Shoot : MLB (hold)

Rotate : A/D

Reset : R



#### Gamepad
Aim : Left Stick

Shoot : A (hold)

Rotate : Left/Right Triggers

Reset : X
